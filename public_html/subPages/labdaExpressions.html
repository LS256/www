Wyrażenia labmda są bardzo przydatną zmianą która pojawiła się w Javie 8. Znakiem rozpoznawczym wyrażeń lambda jest strzałka ->
Proste wyrażenie lambda możemy zapisać tak: 

(a, b) -> a+b;

Odnosząc lambdy do programowania obiektowego, możemy o nich myśleć jak o klasach tymczasowych zawierających jedną metodę. Lambdy to obiekty, zawierające fragment kodu: funkcję, a także specyficzne atrybuty i parametry ważne dla nich (środowisko, w ramach którego operuje funkcja). Dzięki lambdom piszemy zwyczajnie mniej kodu, nie wierzysz – zobacz:

JButton closeButton = new JButton("Close");

//	tradycyjne rozwiązanie
closeButton.addActionListener(new ActionListener(){
	@Override
	public void actionPerformed(ActionEvent e) {
		System.exit(0);	
	}
});

//	Z zastosowaniem Lambda
closeButton.addActionListener(action -> System.exit(0));

W uproszczeniu wyrażenia lambda to funkcje anonimowe. Nie nazywałbym ich metodami, nie przynależą do konkretnej klasy jak metody w języku Java. Wyrażenie lambda nie posiada także nazwy, nie muszą być zdefiniowane typy parametrów, nie muszą być nawet przekazane parametry, posiada ciało i może zwracać konkretny typ lub nie. Ciekawostką jest to, że lambda może być przesłana do metody jako parametr lub przypisana do zmiennej.

Lambdy to także bardzo dobre narzędzie które możemy wykorzystać zamiast pętli:

String[] a = {"cat", "dog", "mouse", "rat", "pig", "rabbit", "hamster", "parrot"};
List animals =  Arrays.asList(a);

// Tradycyjna pętla
for (String animal : animals) {
     System.out.print(animal + "; ");
}


//  Wyrażenie lambda
animals.forEach((animal) -> System.out.print(animal + "; "));


Dzięki lambdom możemy też tworzyć wątki:
//podejście klasyczne
new Thread(new Runnable() {
    @Override
    public void run() {
        System.out.println("Hello world!");
    }
}).start();

//wyrażenie lambda
new Thread(() -> System.out.println("Hello world!")).start();

Przy omawianiu lambd nie sposób nie wspomnieć o strumieniach (stream) które reprezentują sekwencje elementów i pozwalają pracować na tych elementach. Operacje te możemy łączyć w kilka jednocześnie z możliwością zwrócenia wyniku lub nie.

List fruits =    Arrays.asList("jabłko", "gruszka", "banan", "sliwka", 
"winogorono", "brzkoskwinia");

fruits.stream().filter(s -> s.toString().startsWith("b"))
.map(s -> s.toString().toUpperCase()).sorted().forEach(System.out::println);

Wynikiem naszego wywołania będzie:

BANAN
BRZOSKWINIA
